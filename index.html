üîí Our Commitment to Privacy

Gramo is committed to protecting your privacy. This privacy policy explains how our Chrome extension handles data and what information (if any) is collected when you use Gramo.

üìä Data Collection

What We DO NOT Collect:

  • No typing data: Gramo does not record, store, or transmit the text you type on any website
  • No browsing history: We do not track which websites you visit
  • No personal information: No names, emails, addresses, or any identifying information
  • No analytics or tracking: No usage statistics, behavior tracking, or user profiling
  • No keystroke logging: We do not monitor or record your keystrokes
  • No location data: We do not access your geographic location
  • No user accounts: Gramo works without requiring any registration or login

What We DO Store (Locally Only):

  • Extension settings: Your preferences such as:
    • Auto-correct on/off status
    • Synonym feature enabled/disabled
    • Toast notification preferences
    • Selected language (e.g., English US, Spanish, etc.)
    • Sites where you've disabled Gramo

Important: All settings are stored locally in your browser using Chrome's storage API. They never leave your device.

üåê Third-Party Services

LanguageTool API (Grammar Checking)

When you manually trigger a grammar check (by pressing Ctrl+Shift+Y and selecting text), Gramo sends the selected text to LanguageTool's public API to analyze grammar and provide suggestions.

  • This happens only when you explicitly request it
  • Only the selected text is sent, not your entire browsing content
  • LanguageTool has its own privacy policy: https://languagetool.org/legal/privacy

Datamuse API (Synonyms)

When you double-click a word to find synonyms (if enabled), Gramo sends that single word to Datamuse API to retrieve synonym suggestions.

  • Only the word you double-clicked is sent
  • Datamuse API information: https://www.datamuse.com/api/

Auto-Correct (100% Local)

The auto-correct feature uses a built-in dictionary of 300+ common typos stored within the extension. No data is sent to any server for typo corrections.

üîê Permissions Explained

Gramo requests the following Chrome permissions:

  • activeTab: To detect text fields on the current page and show correction popups
  • scripting: To inject the correction interface into web pages
  • storage: To save your settings locally in your browser
  • tabs: To identify website domains for site-specific settings
  • Host permissions (<all_urls>): To work on any website you visit

These permissions are used ONLY for Gramo's core functionality and never for data collection or tracking.
